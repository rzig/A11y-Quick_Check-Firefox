Help and Instructions

Web Accessibility Extension Features

The web accessibility extension provides instant feedback to web developers and content creators in finding potential high level accessibility issues. Structured into discernible tabs, each focuses on critical aspects ranging from content organisation to user experience.

Content and Structure Tab

Evaluates website's content organisation.

  • Proper heading hierarchy.
  • Correct list usage (Ordered, Unordered, Description).
  • HTML 'hgroup' element use.
  • Grouping of related form controls.

Help for check in the Content and Structure Tab

Has Heading Mark-up
Ensure proper heading markup and heading hierarchy.
No Heading Mark-up
Identifies text that is at least 18px and bold, or greater than 24px that is not marked up as a heading.
Text-Level Semantics
Evaluate the use of text-level semantics, such as strong and italic, to enhance content structure and meaning.
Lists
Examine the implementation of lists to ensure content is structured logically. Check if description lists are correctly used for key-value groups.
HTML hgroup
Evaluate the use of the HTML hgroup element for organising heading content.
Related form controls
Verify the proper grouping of related form controls.

Graphic Media Tab

Reviews accessibility of visuals.

  • Validates image ALT attributes.
  • Identifies decorative images and SVGs.
  • Exposes the accessible name for images and SVGs.
  • Examines CSS background images.

Help for check in the Usability and Readability Tab

Target Size
Evaluate the touch size of interactive elements to ensure they are large enough for easy interaction.
Text Spacing
Evaluate if text spacing can be changed for improved readability.
Reading order
Remove page styles (CSS) to review the reading order for content comprehension.
Focus Order
Display focus order count for interactive elements.
ARIA-LABEL
Check for ARIA-LABEL text and elements that cannot be named by the author.

Usability and Readability Tab

Focuses on the user's experience and content clarity.

  • Checks touch size of interactive elements (24px, 44px, 48px).
  • Evaluates changeability of text spacing.
  • Removes page styles for reading order review.
  • Displays focus order for interactive components.
  • Reviews ARIA-LABEL use.

Additional Info:

  • Each feature comes with a help link leading to /extension/ui/help.html.
  • Many features have scripts and CSS for custom checks.
