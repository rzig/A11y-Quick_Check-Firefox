A11y Quick Check

Version: 2501.2024.06.01

Introduction

A11y Quick Check is a testing tool that exposes accessibility issues to provide a high level check to help meet the requirements of the WCAG guidelines and coding best practices.

Version: 2501.2024.05

  • A new summary panel that describes each check, complete with links to related standards and guidelines. The new panel can be minimised, or removed. If, removed,the feature panel will reappear when the check is rerun.
  • Validate Tabindex Usage: This new check aims to ensure proper use of the tabindex attribute. It highlights elements with a tabindex of 0 or higher, promoting best practices in web accessibility and warning against values above 0 to prevent potential accessibility barriers.
  • Duplicate ID Values: Identifying duplicate IDs that affect ARIA attributes and HTML label[for] tags, this check ensures that each ID is unique and correctly used.
  • Enhanced messaging for clarity and to provide additional useful information.
  • Resolved minor bugs and false positives as reported by our dedicated users.
