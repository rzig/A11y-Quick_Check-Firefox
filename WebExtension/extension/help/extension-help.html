A11y Quick Check

Version: 2501.2024.03.04

Introduction

A11y Quick Check is a testing tool that exposes accessibility issues to provide a high level check to help meet the requirements of the WCAG guidelines and coding best practices.

Version: 2501.2024.03.04 update

  • Provides feedback when role presentation or role none is placed on <svg>.
Version: 2501.2024.03.03 update
  • Changed the <title> element to report as a description within an SVG and issue a specific message when it exists.

Version: 2501.2024.03.01 update

  • Update to SVG images check. Now checks for naming via <title>, <text> and the title attribute.
  • Updated DL check which was not reporting correctly when DL is a child of UL > LI or OL LI.

Version: 2501.2024.02.24 update

  • Bug fix where the accessible name of input using 'label for' was being reported as 'Name from none'.

Disclaimer

Every effort has been made to ensure the checks accurately identify both invalid code practices and failures in meeting the WCAG Guidelines. However, some false positive reporting may occur.
