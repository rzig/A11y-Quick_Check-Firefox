A11y Quick Check

Version: 2501.2023.12.19

Introduction

A11y Quick Check is a testing tool that exposes accessibility issues to provide a high level check to help meet the requirements of the WCAG guidelines and coding best practices.

Known false/positives

  • Target-Size when there are two or more adjacent targets with insufficient spacing, the tool is incorrectly identifying the last element as having sufficient spacing.

Disclaimer

All attempts have been made to ensure the checks accurately identifies both invalid code practices and failures in meeting the WCAG Guidelines.
